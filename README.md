# ma-Json - JSON 智能验证与修复工具

ma-Json 是一个功能强大的本地 Web 工具，用于验证、格式化和修复 JSON 数据。除了传统的 JSON 校验功能外，它还集成了大模型（LLM）能力，能够智能识别并修复复杂的 JSON 语法错误。

## ✨ 主要功能

- **JSON 验证与格式化**：
  - 实时检测 JSON 语法错误。
  - 支持语法高亮显示。
  - 提供格式化（美化）后的 JSON 视图。

- **智能修复**：
  - **本地自动修复**：针对常见的简单错误（如末尾逗号、引号缺失等）提供一键修复建议。
  - **AI 智能修复**：集成大模型接口，可智能分析并修复复杂的 JSON 结构问题。

- **实用工具**：
  - **压缩**：一键压缩 JSON 数据，去除空格和换行。
  - **清空/复制**：便捷的操作按钮。

- **安全代理**：
  - 内置 Node.js 代理服务器，解决跨域问题并保护 API 调用安全。

## 🚀 快速开始

ma-Json 旨在提供开箱即用的体验。

### 方式一：纯静态使用（无需 Node.js）

如果您只需要使用 **JSON 验证、格式化、压缩** 以及 **本地自动修复** 功能，您可以直接双击打开项目目录下的 `index.html` 文件即可使用，无需安装任何环境。

*注意：在此模式下，"AI 智能修复" 功能可能因浏览器跨域限制而无法使用，除非您配置的大模型接口支持 CORS。*

### 方式二：完整功能（推荐）

如果您需要使用完整的 **AI 智能修复** 功能，建议安装 Node.js 环境，以便启动本地代理服务解决接口跨域问题。

#### 前置要求

- 请确保您的系统已安装 [Node.js](https://nodejs.org/)。

#### 安装与运行

1. **启动工具**
   - **Windows 用户**：双击运行目录下的 `启动工具.bat`。
   - **命令行启动**：在项目根目录下运行：
     ```bash
     node launcher.js
     ```

2. **访问工具**
   启动成功后，浏览器将自动打开 `http://localhost:8081`。

## ⚙️ 配置说明

点击页面右上角的 **设置 (⚙️)** 按钮，可以配置大模型接口参数：

- **接口地址 (URL)**：大模型的 API 端点
- **API Key**：用于鉴权的 API 密钥。
- **模型名称 (Model)**：指定使用的模型版本。

## 🛠️ 技术栈

- **前端**：HTML5, CSS3, Vanilla JavaScript
- **后端**：Node.js (原生 http/https 模块)
